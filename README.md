# 🚗 CampusRide

**CampusRide** is a smart ride-sharing platform built exclusively for college students. It helps students find, offer, and share rides within the campus community in a safe, affordable, and convenient way.

> Built for students. Trusted by students.

---

## ✨ Features

- 🎓 Student-only ride sharing
- 📍 Live map integration for pickup and drop locations
- 🔍 Smart ride matching
- 🚘 Offer or request a ride
- 💰 Cost splitting between passengers
- 📱 Fully responsive design
- ⚡ Fast and modern user experience
- 🔒 Secure form validation

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/campusride.git
cd campusride
```

### Install dependencies

Using Bun

```bash
bun install
```

or npm

```bash
npm install
```

### Start the development server

```bash
bun run dev
```

or

```bash
npm run dev
```

## 💡 Why CampusRide?

Students often struggle to find affordable and trustworthy transportation for daily commutes. CampusRide solves this by creating a verified ride-sharing network exclusively for college students, making travel safer, cheaper, and more convenient.
